# IITB CS 333 xv6 Operating System
This repository contains various improvements to the xv6 kernel as part of the CS 333 Operating Systems course. Each subdirectory has a separate README file describing the extensions implemented for that particular lab.

## Instructions for running xv6
Inside the `xv6-public` directory, run the following commands to start the QEMU-based xv6 emulator:
```bash
$ make clean
$ make qemu-nox
```
